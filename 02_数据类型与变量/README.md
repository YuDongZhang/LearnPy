# 第二章：数据类型与变量

## 本章目标
- 掌握 Python 的基本数据类型
- 学会变量的定义和使用
- 理解类型转换
- 掌握基本运算符

---

## 1. 变量

变量是存储数据的容器。Python 中定义变量不需要声明类型：

```python
name = "Python"    # 字符串
age = 30           # 整数
price = 3.14       # 浮点数
is_active = True   # 布尔值
```

### 变量命名规则
- 只能包含字母、数字、下划线
- 不能以数字开头
- 区分大小写
- 不能使用 Python 关键字

---

## 2. 基本数据类型

| 类型 | 名称 | 示例 |
|------|------|------|
| int | 整数 | 1, 100, -50 |
| float | 浮点数 | 3.14, -0.5 |
| str | 字符串 | "hello", 'world' |
| bool | 布尔值 | True, False |

---

## 3. 类型转换

```python
# 字符串转整数
num = int("123")

# 整数转字符串
text = str(123)

# 整数转浮点数
decimal = float(10)
```

---

## 4. 运算符

### 算术运算符
| 运算符 | 说明 | 示例 |
|--------|------|------|
| + | 加 | 1 + 2 = 3 |
| - | 减 | 5 - 2 = 3 |
| * | 乘 | 2 * 3 = 6 |
| / | 除 | 6 / 2 = 3.0 |
| // | 整除 | 7 // 2 = 3 |
| % | 取余 | 7 % 2 = 1 |
| ** | 幂 | 2 ** 3 = 8 |

### 比较运算符
`==` `!=` `>` `<` `>=` `<=`

### 逻辑运算符
`and` `or` `not`
