# 第三十二章：自然语言处理 (NLP)

## 本章目标

- 了解自然语言处理的基本概念
- 掌握文本预处理技术
- 学会使用 RNN/LSTM 处理文本
- 理解 Transformer 架构
- 掌握 BERT、GPT 等预训练模型的使用

## NLP 简介

自然语言处理 (Natural Language Processing, NLP) 是人工智能与语言学的交叉学科，研究如何让计算机理解和生成人类语言。
