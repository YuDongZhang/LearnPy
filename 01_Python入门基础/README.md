## 1. Python 简介

Python 是一门简洁、易学、功能强大的编程语言。它的特点是：
- 语法简洁，接近自然语言
- 跨平台，可在 Windows、Mac、Linux 上运行
- 拥有丰富的第三方库

---

## 2. 第一个程序

传统上，学习编程的第一个程序都是输出 "Hello, World!"：

```python
print("Hello, World!")
```

`print()` 是 Python 中最常用的函数，用于在屏幕上输出内容。

---

## 3. 注释

注释是代码中的说明文字，不会被执行。

### 单行注释
使用 `#` 开头：
```python
# 这是一个单行注释
print("你好")  # 这也是注释
```

### 多行注释
使用三个引号：
```python
'''
这是多行注释
可以写很多行
'''
```

---

## 4. 基本输出

```python
print("你好，Python")  # 输出字符串
print(123)             # 输出数字
print(3.14)            # 输出小数
```

---

## 5. 基本输入

使用 `input()` 函数获取用户输入：

```python
name = input("请输入你的名字：")
print("你好，" + name)
```

**注意**：`input()` 返回的永远是字符串类型。

---

## 6. 代码缩进

Python 使用缩进来表示代码块，通常使用 4 个空格：

```python
if True:
    print("条件为真")  # 缩进4个空格
```

缩进错误会导致程序无法运行！
