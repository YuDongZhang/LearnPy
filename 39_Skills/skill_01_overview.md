# 第1节：Skills是什么？

本节介绍Skills的基本概念，帮助你理解Skills是什么以及为什么需要它。

---

## 什么是Skills？

**Skills（技能系统）** 是一种让AI助手（如Claude、Trae）扩展能力的机制。

简单来说，Skill就像给AI安装一个"插件"，让它学会新技能。

### 生活中的类比

| 概念 | 例子 |
|------|------|
| AI本身 | 一个大学生 |
| Skill | 一本专业教材/技能培训 |
| 使用Skill | 大学生学习后掌握新技能 |

---

## 为什么需要Skills？

### 没有Skill的情况
```
用户: 帮我审查代码
AI: 我不会审查代码，我只擅长对话
```

### 有Skill的情况
```
用户: 帮我审查代码
AI: (调用CodeReviewer Skill)
→ 分析代码质量
→ 检查安全问题
→ 给出改进建议
→ 评分: 85/100
```

---

## Skill的核心特性

### 1. 可扩展性
- 随时添加新技能
- 不需要改AI核心代码

### 2. 模块化
- 每个Skill只做一件事
- 便于测试和维护

### 3. 可复用
- 一次编写，多次使用
- 大家都能用

### 4. 可组合
- 多个Skill配合工作
- 解决复杂问题

---

## 总结

- **Skill = AI的插件**
- **让AI从"什么都不会"变成"什么都能干"**
- **核心思想：模块化、可扩展、可复用**
