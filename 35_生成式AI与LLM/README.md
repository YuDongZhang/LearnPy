## 生成式AI简介

生成式AI (Generative AI) 是人工智能的一个重要分支，能够创建新的内容，如文本、图像、音频、视频等。近年来，随着大语言模型(LLM)的快速发展，生成式AI在各个领域展现出强大的能力。
