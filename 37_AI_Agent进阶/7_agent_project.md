# 7. 进阶实战项目

## 项目：AI研究团队

用LangGraph构建一个多Agent协作的研究团队，自动完成"研究→写作→审核→修订"的完整流程。

## 架构

```
用户输入主题
    ↓
研究员节点 → 搜索收集信息
    ↓
写手节点 → 撰写文章
    ↓
审核员节点 → 审核质量
    ↓
判断 ─→ 需要修改 → 修订节点 → 回到审核
    └→ 通过 → 输出最终文章
```

## 技术要点

1. 使用LangGraph的StateGraph构建工作流
2. 三个Agent角色：研究员、写手、审核员
3. 条件边实现"审核不通过→修订→重新审核"的循环
4. 设置最大迭代次数防止死循环
5. 状态持久化支持中断恢复

## 运行方式

```bash
python 7_agent_project.py
```

输入研究主题，系统自动完成研究、写作、审核流程。

## 扩展方向

- 加入人类审核节点（Human-in-the-Loop）
- 并行多团队协作（技术团队+商业团队）
- 接入真实搜索工具（DuckDuckGo）
- 加入向量数据库做长期记忆
- 用FastAPI包装为Web服务
