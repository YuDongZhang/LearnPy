# 第十九章：项目实战

## 本章目标
- 综合运用 Python 基础知识
- 完成一个完整的命令行工具
- 掌握项目开发的完整流程

---

## 1. 项目介绍

本章将创建一个**命令行待办事项管理工具** (Todo List CLI)，涵盖：

- 添加、查看、删除、标记完成待办事项
- 数据持久化存储（JSON 文件）
- 友好的命令行交互界面

---

## 2. 功能特性

| 功能 | 说明 |
|------|------|
| 添加任务 | 添加新的待办事项 |
| 列出任务 | 查看所有待办事项 |
| 完成任务 | 标记任务为已完成 |
| 删除任务 | 删除不需要的任务 |
| 数据持久化 | 保存到本地 JSON 文件 |

---

## 3. 示例文件

| 文件 | 内容 |
|------|------|
| `todo.py` | 待办事项管理工具主程序 |

---

## 4. 运行方式

```bash
python todo.py --help
python todo.py add "学习 Python"
python todo.py list
python todo.py done 1
python todo.py delete 2
```
