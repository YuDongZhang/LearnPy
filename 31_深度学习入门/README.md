# 第三十一章：深度学习入门

## 本章目标

- 了解深度学习的基本概念
- 掌握神经网络的工作原理
- 学会使用 TensorFlow 和 PyTorch 框架
- 动手实践简单的神经网络项目

## 深度学习简介

深度学习 (Deep Learning) 是机器学习的一个分支，使用多层神经网络来学习数据的表征。
