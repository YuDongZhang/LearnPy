# 4. 面向对象

## 类定义

```java
// Java
public class Dog {
    private String name;
    private int age;

    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String bark() {
        return name + " says woof!";
    }

    @Override
    public String toString() {
        return "Dog(" + name + ", " + age + ")";
    }
}
```

```python
# Python
class Dog:
    def __init__(self, name, age):
        self.name = name    # 没有private，约定_前缀表示私有
        self.age = age

    def bark(self):         # 第一个参数必须是self
        return f"{self.name} says woof!"

    def __str__(self):      # 等价于toString()
        return f"Dog({self.name}, {self.age})"
```

## 关键差异

| 特性 | Java | Python |
|------|------|--------|
| 构造函数 | `ClassName()` | `__init__(self)` |
| this | `this` | `self`（必须显式写） |
| 访问控制 | `public/private/protected` | 约定：`_`私有，`__`强私有 |
| toString | `toString()` | `__str__()` |
| equals | `equals()` | `__eq__()` |
| hashCode | `hashCode()` | `__hash__()` |
| getter/setter | 方法 | `@property` 装饰器 |
| static方法 | `static` | `@staticmethod` |
| 抽象类 | `abstract class` | `from abc import ABC` |
| 接口 | `interface` | 没有，用ABC或Protocol |

## 继承

```java
// Java: 单继承
public class Puppy extends Dog {
    public Puppy(String name) {
        super(name, 0);
    }
}
```

```python
# Python: 支持多继承
class Puppy(Dog):
    def __init__(self, name):
        super().__init__(name, 0)

# 多继承
class GuideDog(Dog, Trainable):
    pass
```

## Property（替代getter/setter）

```java
// Java
private int age;
public int getAge() { return age; }
public void setAge(int age) { this.age = age; }
```

```python
# Python
class Person:
    def __init__(self, age):
        self._age = age

    @property
    def age(self):
        return self._age

    @age.setter
    def age(self, value):
        if value < 0:
            raise ValueError("年龄不能为负")
        self._age = value

p = Person(25)
p.age        # 调用getter，像访问属性一样
p.age = 30   # 调用setter
```

## dataclass（类似Java的record）

```python
from dataclasses import dataclass

@dataclass
class User:
    name: str
    age: int
    email: str = ""

# 自动生成__init__、__repr__、__eq__
u = User("张三", 25)
```

## 关键记忆点

1. **self必须显式写** — 每个方法第一个参数是self
2. **没有真正的private** — `_`是约定，`__`会名称改写
3. **支持多继承** — Java只能单继承+接口
4. **`@property`** — 比getter/setter优雅
5. **`@dataclass`** — 类似Java record，省大量样板代码
6. **魔术方法** — `__init__`, `__str__`, `__eq__` 等，对应Java的特殊方法
