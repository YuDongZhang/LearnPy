## 阅读方式

## 简单总结

- 变量无需声明：`age = 15`
- 不用分号，换行就是语句结尾
- 函数用 `def` 加冒号 `:`
- 缩进代替大括号 `{}`，缩进就是作用域
- 没有基本类型，一切皆对象
- 动态类型，变量不绑定类型

## Java人最容易踩的坑

- `True/False/None` 首字母大写，不是 `true/false/null`
- 用 `and/or/not`，不是 `&&/||/!`
- 字符串比较直接用 `==`，不需要 `.equals()`
- 方法第一个参数必须写 `self`，Java的 `this` 是隐式的
- `elif` 不是 `else if`
- 没有 `++/--`，用 `+= 1`
- 除法 `/` 默认返回浮点数，整除用 `//`
- `print()` 是函数不是语句，要加括号
