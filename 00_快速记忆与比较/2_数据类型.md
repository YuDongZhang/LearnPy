# 2. 数据类型对比

## 类型映射

| Java | Python | 说明 |
|------|--------|------|
| `int` | `int` | Python的int无上限，不会溢出 |
| `long` | `int` | 同上，Python不区分int/long |
| `float/double` | `float` | Python只有float（64位） |
| `boolean` | `bool` | `True/False`（大写） |
| `char` | `str` | Python没有char，单字符也是str |
| `String` | `str` | 不可变，和Java一样 |
| `int[]` | `list` | Python的list可以混合类型 |
| `ArrayList` | `list` | 就是list |
| `HashMap` | `dict` | `{"key": "value"}` |
| `HashSet` | `set` | `{1, 2, 3}` |
| `null` | `None` | |

## 类型检查

```java
// Java
if (obj instanceof String) { ... }
obj.getClass().getName()
```

```python
# Python
if isinstance(obj, str): ...
type(obj)  # <class 'str'>
```

## 数字

```java
// Java
int a = Integer.MAX_VALUE;  // 2147483647
long b = Long.MAX_VALUE;
```

```python
# Python: int没有上限
a = 2 ** 100  # 直接算，不会溢出
b = 10 ** 1000  # 也没问题
```

## 字符串

```java
// Java
String s = "hello";
s.charAt(0);          // 'h'
s.substring(1, 3);    // "el"
s.equals("hello");    // true
```

```python
# Python
s = "hello"
s[0]          # 'h'
s[1:3]        # 'el'（切片）
s == "hello"  # True（直接用==比较内容）
```

### 切片（Java没有的）

```python
s = "Hello World"
s[0:5]    # "Hello"
s[6:]     # "World"
s[-5:]    # "World"
s[::-1]   # "dlroW olleH"（反转）
```

## 类型注解（可选）

Python 3.5+支持类型注解，但不强制：
```python
def greet(name: str, age: int) -> str:
    return f"Hello {name}, you are {age}"
```
类似Java的类型声明，但Python运行时不检查。

## 关键记忆点

1. **Python的int没有上限** — 不用担心溢出
2. **没有char** — 单字符就是长度为1的str
3. **用==比较字符串** — 不需要`.equals()`
4. **切片** — `s[start:end:step]` 非常强大
5. **list可以混合类型** — `[1, "hello", True]` 合法
6. **类型注解是可选的** — 写了更清晰，不写也能跑
