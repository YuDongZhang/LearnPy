# 6. 异常处理与文件IO

## 异常处理

```java
// Java
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("除零错误: " + e.getMessage());
} catch (Exception e) {
    System.out.println("其他错误");
} finally {
    System.out.println("清理");
}
```

```python
# Python: except不是catch，没有throws
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"除零错误: {e}")
except Exception as e:
    print("其他错误")
finally:
    print("清理")
```

### 对照表

| Java | Python |
|------|--------|
| `try` | `try` |
| `catch` | `except` |
| `finally` | `finally` |
| `throw` | `raise` |
| `throws` | 没有（不需要声明） |
| `try-with-resources` | `with` 语句 |

## 自定义异常

```java
public class MyException extends Exception {
    public MyException(String msg) { super(msg); }

```

```python
class MyException(Exception):

raise MyException("出错了")
```

## 文件IO

```java
// Java: try-with-resources
try (BufferedReader br = new BufferedReader(new FileReader("file.txt"))) {
    String line;
    while ((line = br.readLine()) != null) {
        System.out.println(line);
    }
}
```

```python
# Python: with语句（等价于try-with-resources）
with open("file.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

### 文件操作对照

| 操作 | Java | Python |
|------|------|--------|
| 读全部 | `Files.readString(path)` | `f.read()` |
| 逐行读 | `BufferedReader.readLine()` | `for line in f` |
| 写文件 | `FileWriter.write()` | `f.write()` |
| 追加 | `new FileWriter(f, true)` | `open(f, "a")` |
| 路径 | `Path.of()` | `pathlib.Path()` |
| 文件存在 | `Files.exists()` | `Path.exists()` |

### 文件模式

| 模式 | 说明 |
|------|------|
| `"r"` | 读（默认） |
| `"w"` | 写（覆盖） |
| `"a"` | 追加 |
| `"rb"` | 二进制读 |
| `"wb"` | 二进制写 |

## JSON

```java
// Java (Jackson)
ObjectMapper mapper = new ObjectMapper();
Map data = mapper.readValue(jsonStr, Map.class);
String json = mapper.writeValueAsString(data);
```

```python
# Python (内置)
import json
data = json.loads(json_str)       # 字符串→对象
json_str = json.dumps(data)       # 对象→字符串
# 文件
with open("data.json") as f:
    data = json.load(f)           # 文件→对象
```

## 关键记忆点

1. **except不是catch** — `except Exception as e`
2. **raise不是throw** — `raise ValueError("msg")`
3. **不需要throws声明** — Python不检查异常
4. **with语句** — 等价于try-with-resources，更简洁
5. **json是内置模块** — 不需要Jackson/Gson
6. **open()指定encoding** — `open(f, encoding="utf-8")`
