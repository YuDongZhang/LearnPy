# 3. 控制流与函数

## if/else

```java
// Java
if (age >= 18) {
    System.out.println("成年");
} else if (age >= 12) {
    System.out.println("青少年");
} else {
    System.out.println("儿童");
}
```

```python
# Python: elif不是else if
if age >= 18:
    print("成年")
elif age >= 12:
    print("青少年")
else:
    print("儿童")
```

## switch → match（Python 3.10+）

```java
// Java
switch (day) {
    case "Mon": System.out.println("周一"); break;
    case "Tue": System.out.println("周二"); break;
    default: System.out.println("其他");
}
```

```python
# Python 3.10+
match day:
    case "Mon": print("周一")
    case "Tue": print("周二")
    case _: print("其他")
```

## for循环

```java
// Java
for (int i = 0; i < 10; i++) { ... }
for (String item : list) { ... }
```

```python
# Python
for i in range(10): ...       # 0到9
for item in list: ...          # 遍历
for i, item in enumerate(list): ...  # 带索引遍历
```

### range对照

| Python | 等价Java |
|--------|---------|
| `range(5)` | `i = 0; i < 5; i++` |
| `range(2, 8)` | `i = 2; i < 8; i++` |
| `range(0, 10, 2)` | `i = 0; i < 10; i += 2` |
| `range(10, 0, -1)` | `i = 10; i > 0; i--` |

## while

```java
while (condition) { ... }
do { ... } while (condition);  // Python没有do-while
```

```python
while condition: ...
# Python没有do-while，用while True + break模拟
```

## 函数

```java
// Java: 必须在类里，必须声明返回类型
public static int add(int a, int b) {
    return a + b;
}
```

```python
# Python: 独立函数，不需要类
def add(a, b):
    return a + b
```

## 默认参数（Java没有）

```python
def greet(name, greeting="你好"):
    return f"{greeting}, {name}"

greet("张三")           # "你好, 张三"
greet("张三", "Hi")     # "Hi, 张三"
```

## 可变参数

```java
public void print(String... args) { }
```

```python
def func(*args, **kwargs):
    # args: 位置参数元组
    # kwargs: 关键字参数字典
    pass

func(1, 2, 3, name="张三")
# args = (1, 2, 3), kwargs = {"name": "张三"}
```

## Lambda

```java
// Java
Function<Integer, Integer> square = x -> x * x;
list.stream().filter(x -> x > 0).collect(Collectors.toList());
```

```python
# Python
square = lambda x: x * x
list(filter(lambda x: x > 0, my_list))
# 更Pythonic的写法：
[x for x in my_list if x > 0]
```

## 关键记忆点

1. **elif** — 不是 `else if`
2. **for in** — 没有C风格for循环，用 `range()`
3. **函数不需要类** — 直接 `def` 定义
4. **默认参数** — Java没有，Python有
5. **`*args, **kwargs`** — 比Java的可变参数更灵活
6. **列表推导式** — `[x for x in list if ...]` 替代stream
