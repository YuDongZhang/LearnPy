# 5. 集合与数据结构

## 类型映射

| Java | Python | 字面量 |
|------|--------|--------|
| `ArrayList<T>` | `list` | `[1, 2, 3]` |
| `LinkedList<T>` | `collections.deque` | `deque([1, 2, 3])` |
| `HashMap<K,V>` | `dict` | `{"a": 1, "b": 2}` |
| `HashSet<T>` | `set` | `{1, 2, 3}` |
| `Tuple`(没有) | `tuple` | `(1, 2, 3)` 不可变 |
| `Stack` | `list` | 用append/pop |
| `Queue` | `collections.deque` | 用append/popleft |

## List

```java
// Java
List<String> list = new ArrayList<>();
list.add("a");
list.get(0);
list.size();
list.remove(0);
list.contains("a");
Collections.sort(list);
```

```python
# Python
lst = []
lst.append("a")
lst[0]
len(lst)
lst.pop(0)       # 或 del lst[0]
"a" in lst
lst.sort()
```

### List操作速查

| 操作 | Java | Python |
|------|------|--------|
| 添加 | `add(x)` | `append(x)` |
| 插入 | `add(i, x)` | `insert(i, x)` |
| 删除 | `remove(i)` | `pop(i)` 或 `del lst[i]` |
| 长度 | `size()` | `len(lst)` |
| 包含 | `contains(x)` | `x in lst` |
| 排序 | `Collections.sort()` | `lst.sort()` 或 `sorted(lst)` |
| 反转 | `Collections.reverse()` | `lst.reverse()` 或 `lst[::-1]` |
| 切片 | 没有 | `lst[1:3]` |
| 合并 | `addAll()` | `lst1 + lst2` |

## Dict（HashMap）

```java
// Java
Map<String, Integer> map = new HashMap<>();
map.put("age", 25);
map.get("age");
map.containsKey("age");
map.getOrDefault("name", "unknown");
map.keySet();
map.values();
for (Map.Entry<String, Integer> e : map.entrySet()) { }
```

```python
# Python
d = {}
d["age"] = 25
d["age"]
"age" in d
d.get("name", "unknown")
d.keys()
d.values()
for k, v in d.items(): ...
```

## Set

```java
Set<Integer> set = new HashSet<>(Arrays.asList(1, 2, 3));
set.add(4);
set.contains(1);
set1.retainAll(set2);  // 交集
```

```python
s = {1, 2, 3}
s.add(4)
1 in s
s1 & s2   # 交集
s1 | s2   # 并集
s1 - s2   # 差集
```

## 列表推导式（替代Stream）

```java
// Java Stream
list.stream()
    .filter(x -> x > 0)
    .map(x -> x * 2)
    .collect(Collectors.toList());
```

```python
# Python列表推导式
[x * 2 for x in lst if x > 0]
```

## 关键记忆点

1. **`in`关键字** — 替代contains/containsKey，通用
2. **切片** — `lst[1:3]`，Java没有
3. **列表推导式** — `[expr for x in lst if cond]`，替代Stream
4. **dict字面量** — `{"key": value}`，比HashMap简洁太多
5. **set运算** — `& | -`，比Java的retainAll/addAll直观
6. **tuple** — 不可变列表，可以做dict的key
