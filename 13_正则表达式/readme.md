# 第十三章：正则表达式

## 本章目标
- 理解正则表达式语法
- 掌握 re 模块使用方法
- 学会模式匹配和替换

---

## 1. 基本语法

| 模式 | 说明 |
|------|------|
| `.` | 任意字符 |
| `\d` | 数字 |
| `\w` | 字母数字下划线 |
| `\s` | 空白字符 |
| `*` | 0次或多次 |
| `+` | 1次或多次 |
| `?` | 0次或1次 |

---

## 2. re 模块

```python
import re
pattern = r"\d+"
match = re.search(pattern, "abc123")
```
